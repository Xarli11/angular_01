Angular
Bibliotecas y frameworks Javascript de código abierto

Las aplicaciones angulares están escritas en TypeScript . TypeScript agrega sintaxis a JavaScript para proporcionar una verificación de tipos sólida. Su entorno de desarrollo integrado (IDE) puede aplicar la verificación de tipo estático para ayudar a detectar errores antes y ayudarlo cuando refactorice más tarde. La sólida verificación de tipos de TypeScript le brinda a su aplicación una base sólida sobre la cual crecer con confianza.
